# Notes App

Notes App is a simple and efficient web application that allows users to create, edit, delete, and manage notes in real time. It provides a clean and user-friendly interface for organizing thoughts, tasks, and important information.

The application stores notes dynamically and ensures quick access and updates without reloading the page. It is designed to improve productivity by helping users keep track of their daily tasks and ideas.

## 🚀 Features
- Create new notes instantly
- Edit and update existing notes
- Delete notes بسهولة (easily)
- Real-time updates without page refresh
- Clean and responsive UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript (DOM Manipulation / Local Storage)

## 📌 Use Case
This app is useful for managing daily tasks, quick reminders, and organizing ideas in a simple and effective way.
